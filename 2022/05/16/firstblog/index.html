title:Windows 安装npm报错

#windows安装npm报错
本篇共解决三个问题：ERR；Run npm install -g npm@8.10.0 to update； hexo command not found）

##1. ERR错误
在安装npm时，遇到第一个问题是一连串的ERR错误，网上主要有两种解决方法，如果you遇到同样问题，可以先尝试如下
解决方法：

  • 删除CUsers{user}下的.npmrc文件

  • 运行命令：npm cache clean –force
    但我尝试两种方法后，安装npm，仍然还存在以下问题：

    解决方法：
    搞了好久，我检查之后，发现是我打开翻墙软件，所以这个报错有可能是pip打不开需要访问的地址。关掉后，问题解决！
